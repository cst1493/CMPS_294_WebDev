Book Inventory Management






This is the inventory database.
You must include a book title in order to add a new entry, as this is how to search for a book.
Note that the book titles are cap sensitive. "Alice in Wonderland" is different than "alice in wonderland".

When you enter, manipulate or wish to see the data, you will be sent to a new tab showing the given data.
For delete and select all, you only need to enter the Title section, as this is a required field.
